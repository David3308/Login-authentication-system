# 1. Requirement Analysis
This system allows a user to log in using a predefined username and password. It checks whether the entered details are correct.
# Functional Requirements
* Accept username and password
* Validate login credentials
* Display login status
# Non-Functional Requirements
* Simple
* Secure enough for basic use
* Console-based.
